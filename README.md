Contribute your styling and animation in it 

This project is an experimental creative coding animation built with canvas-sketch. The goal was to create a smooth, modern loading screen using animated geometric shapes and procedural graphics.

The current version is functional, but I'm still struggling to achieve the smooth transition timing and animation behavior I have in mind. If you have experience with canvas-sketch, the HTML5 Canvas API, or animation timing, I'd love your help!

Contributions Welcome  
If you know how to improve:

Smooth animation transitions
Rotation and scaling timing
Better easing and interpolation
Performance optimization
Loading screen aesthetics
Overall visual quality

feel free to fork the repository, submit a pull request, or share your ideas through an issue. Every contribution is appreciated and will help make this project better.
